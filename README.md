# Hospital-Web

Sistema web hospitalario enfocado en la gestión de pacientes, citas médicas e historial clínico.

⸻

Descripción

Hospital Web es una plataforma desarrollada como proyecto final web 1, diseñada para facilitar la administración de un hospital mediante un sistema moderno, intuitivo y responsive.

El sistema permite gestionar:

* Doctores
* Pacientes
* Citas médicas
* Historial clínico
* Usuarios y autenticación
* Panel administrativo

El objetivo es ofrecer una experiencia similar a plataformas hospitalarias reales utilizando tecnologías web.

⸻
